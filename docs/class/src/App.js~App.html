    App — a container component for the currency converter.

    Responsible for converting between base and quote currencies, and displaying so-called "currency widgets". The first currency widget in the list defines what the base currency is, subsequent widgets map to quote currencies.

    See:

    • https://www.investopedia.com/terms/b/basecurrency.asp
